# Day01_HW1

## 提交关于课堂教授有关指令

### git config.png

````shell
$git config --global user.name
$git config --global user.email 
$git config --list #或者使用vim .gitconfig
````



### git clone.png

```shell
$git clone http://...   #克隆仓库
$git add .							#添加当前文件夹所有文件到准备区
$git commit -m ""				#添加提交信息
$git push								#提交
$git pull								#拉取仓库文件
$git status							#当前分支状态
```



### g++ -o.png

```shell
$g++ hello.cpp -o hello		#将hello.cpp编译为hello.o
```
